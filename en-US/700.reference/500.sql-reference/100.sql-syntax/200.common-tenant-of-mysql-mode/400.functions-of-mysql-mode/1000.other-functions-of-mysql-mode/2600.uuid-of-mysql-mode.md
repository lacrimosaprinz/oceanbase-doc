# UUID

## Syntax

```sql
UUID()
```

## Purpose

`UUID()` generates a globally unique ID.

## Examples

```sql
obclient> SELECT UUID();
+--------------------------------------+
| UUID()                               |
+--------------------------------------+
| eef95d17-e944-11eb-824a-ca1fa9040697 |
+--------------------------------------+
1 row in set
```
