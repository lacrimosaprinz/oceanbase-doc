# DAYOFWEEK

## Syntax

```sql
DAYOFWEEK(date)
```

## Purpose

`DAYOFWEEK()` returns the ordinal number of the day in the week of `date`, starting on Sunday.

## Examples

```sql
obclient> SELECT DAYOFWEEK('2021-07-15');
+-------------------------+
| DAYOFWEEK('2021-07-15') |
+-------------------------+
|                       5 |
+-------------------------+
1 row in set
```
