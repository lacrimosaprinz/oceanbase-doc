# MONTH

## Syntax

```javascript
MONTH(date)
```

## Purpose

`MONTH()` returns the month of `date`.

## Examples

```javascript
obclient> SELECT MONTH('2008-02-03');
+---------------------+
| MONTH('2008-02-03') |
+---------------------+
|                   2 |
+---------------------+
1 row in set
```
