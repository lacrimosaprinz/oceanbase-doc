# CURRENT_DATE

## Syntax

```sql
CURRENT_DATE()
CURRENT_DATE
```

## Purpose

`CURRENT_DATE` returns the current date without the time part. The function is an alias for [CURDATE](../100.date-and-time-functions-of-mysql-mode/300.curdate-of-mysql-mode.md).

## Examples

```javascript
obclient> SELECT CURRENT_DATE;
+--------------+
| CURRENT_DATE |
+--------------+
| 2021-08-23   |
+--------------+
1 row in set

obclient> SELECT CURRENT_DATE();
+----------------+
| CURRENT_DATE() |
+----------------+
| 2021-08-23     |
+----------------+
1 row in set
```
