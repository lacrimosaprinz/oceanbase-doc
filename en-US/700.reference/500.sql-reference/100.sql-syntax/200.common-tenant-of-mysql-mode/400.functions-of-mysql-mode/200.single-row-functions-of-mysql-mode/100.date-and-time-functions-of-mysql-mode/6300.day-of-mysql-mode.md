# DAY

## Syntax

```sql
DAY(date)
```

## Purpose

`DAY()` returns the day of a month in `date`, ranging from 1 to 31. `DAY()` is a synonym for `DAYOFMONTH()`.

## Examples

```sql
obclient> SELECT DAY('2008-12-31');
+--------------------------+
| DAY('2008-12-31') |
+--------------------------+
|                       31 |
+--------------------------+
```
