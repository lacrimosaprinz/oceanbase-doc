# DATE

## Syntax

```sql
DATE(expr)
```

## Purpose

`DATE()` returns the date of the time or datetime expression `expr`.

## Examples

```sql
obclient> SELECT DATE('2021-07-14 19:12:03');
+-----------------------------+
| DATE('2021-07-14 19:12:03') |
+-----------------------------+
| 2021-07-14                  |
+-----------------------------+
1 row in set
```
