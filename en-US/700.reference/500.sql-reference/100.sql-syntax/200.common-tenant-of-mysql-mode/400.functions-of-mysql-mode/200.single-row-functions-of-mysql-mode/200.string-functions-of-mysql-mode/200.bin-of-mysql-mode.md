# BIN

## Syntax

```sql
BIN(N)
```

## Purpose

`BIN()` returns the binary form of the number `N`. This function is equivalent to `CONV(N,10,2)`.

If `N` is `NULL`, `NULL` is returned.

## Examples

```sql
obclient> SELECT BIN(12);
+---------+
| BIN(12) |
+---------+
| 1100    |
+---------+
1 row in set
```
