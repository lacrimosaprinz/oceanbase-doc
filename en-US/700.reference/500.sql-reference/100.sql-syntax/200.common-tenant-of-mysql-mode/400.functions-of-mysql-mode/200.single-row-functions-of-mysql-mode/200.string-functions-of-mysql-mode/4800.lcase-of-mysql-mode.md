# LCASE

## Syntax

```sql
LCASE(str)
```

## Purpose

`LCASE()` converts uppercase letters in a string into lowercase letters.

The function is a synonym for the `LOWER()` function.

## Examples

```sql
obclient> SELECT LCASE('OBCLIENTHELLO! ');
+----------------------------+
| LCASE('OBCLIENTHELLO! ')   |
+----------------------------+
| obclienthello!             |
+----------------------------+
1 row in set
```
