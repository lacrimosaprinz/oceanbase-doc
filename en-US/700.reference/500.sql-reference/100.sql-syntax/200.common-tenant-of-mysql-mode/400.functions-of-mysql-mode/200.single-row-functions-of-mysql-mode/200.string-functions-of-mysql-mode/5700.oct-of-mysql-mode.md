# OCT

## Syntax

```sql
OCT(N)
```

## Purpose

`OCT()` returns the octal value of a numeric value `N`. This function is equivalent to the `CONV(N,10,8)` function.

If `N` is `NULL`, `NULL` is returned.

## Examples

```sql
obclient> SELECT OCT(12);
+---------+
| OCT(12) |
+---------+
| 14      |
+---------+
1 row in set
```
