# OCTET_LENGTH

## Syntax

```sql
OCTET_LENGTH(str)
```

## Purpose

`OCTET_LENGTH()` is a synonym for `LENGTH()`. 

## Examples

```sql
obclient> SELECT OCTET_LENGTH('International'), OCTET_LENGTH('hello');
+------------------------+-----------------------+
| OCTET_LENGTH('International')   | OCTET_LENGTH('hello') |
+------------------------+-----------------------+
|                      13 |                    5 |
+------------------------+-----------------------+
1 row in set
```
