# RADIANS

## Syntax

```sql
RADIANS(X)
```

## Purpose

`RADIANS()` converts an angle `X` from the measure of degrees to radians and returns the radians equivalent of the angle. `Ï€` radians are equivalent to `180` degrees.

## Examples

```sql
obclient> SELECT RADIANS(-10), RADIANS(0), RADIANS(10), RADIANS(-0.5), RADIANS(0.5);
```
