# TAN

## Syntax

```sql
TAN(X)
```

## Purpose

`TAN()` returns the tangent of the specified X radian.

## Examples

```sql
obclient> SELECT TAN(PI());
+-------------------------+
| TAN(PI())               |
+-------------------------+
| -1.2246467991473532e-16 |
+-------------------------+
1 row in set
```
