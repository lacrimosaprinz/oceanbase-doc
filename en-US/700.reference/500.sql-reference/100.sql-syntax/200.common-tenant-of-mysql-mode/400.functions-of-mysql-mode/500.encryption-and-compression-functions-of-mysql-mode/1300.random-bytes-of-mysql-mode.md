# RANDOM_BYTES

## Syntax

```sql
RANDOM_BYTES(len)
```

## Purpose

`RANDOM_BYTES()` returns a random binary string with a length in bytes specified by the `len` parameter. The value range of `len` is [1,1024].

## Examples

```sql
obclient [test]> select RANDOM_BYTES(2);
+-----------------+
| RANDOM_BYTES(2) |
+-----------------+
| H               |
+-----------------+
1 row in set
```
