# SCHEMA

## Syntax

```sql
SCHEMA()
```

## Purpose

`SCHEMA()` returns the name of the current schema. This function is a synonym for `DATABASE()`.

## Examples

Query the name of the current schema.

```sql
obclient> SELECT SCHEMA();
+----------+
| SCHEMA() |
+----------+
| test     |
+----------+
1 row in set
```
