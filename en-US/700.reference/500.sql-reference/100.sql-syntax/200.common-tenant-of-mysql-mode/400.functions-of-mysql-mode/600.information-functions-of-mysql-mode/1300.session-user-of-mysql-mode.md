# SESSION_USER

## Syntax

```sql
SESSION_USER()
```

## Purpose

`SESSION_USER()` returns the current username and hostname. The return value uses the utf8mb4 character set.

This function is a synonym for `USER()`.

## Examples

```sql
obclient> SELECT SESSION_USER();
+---------------------+
| SESSION_USER()      |
+---------------------+
| root@100.0.0.1 |
+---------------------+
1 row in set
```
