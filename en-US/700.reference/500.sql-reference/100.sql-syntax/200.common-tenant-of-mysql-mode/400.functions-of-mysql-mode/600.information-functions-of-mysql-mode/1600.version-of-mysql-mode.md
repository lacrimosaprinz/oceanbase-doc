# VERSION

## Syntax

```sql
VERSION()
```

## Purpose

`VERSION()` returns the MySQL-compatible version of OceanBase Database. The return value uses the utf8mb4 character set.

## Examples

```sql
obclient> SELECT VERSION();
+---------------------------+
| VERSION()                 |
+---------------------------+
| 5.7.25-OceanBase-v4.1.0.0 |
+---------------------------+
1 row in set
```
