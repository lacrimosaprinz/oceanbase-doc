# USE

## Purpose

You can use this statement to switch the database in use.

## Syntax

```sql
USE database_name
```

## Parameters

| Parameter | Description |
|---------------|-----------|
| database_name | The name of the database to use.  |

## Examples

Switch the current database.

```sql
obclient> USE test
Database changed
```
