# USER

## Purpose

`USER` returns the name of the current logged-on user.

## Syntax

```sql
USER
```

## Return type

The return type is `VARCHAR2`.

## Examples

Return the name of the current user.

```sql
obclient> SELECT USER FROM DUAL;
+------+
| USER |
+------+
| SYS  |
+------+
1 row in set
```
