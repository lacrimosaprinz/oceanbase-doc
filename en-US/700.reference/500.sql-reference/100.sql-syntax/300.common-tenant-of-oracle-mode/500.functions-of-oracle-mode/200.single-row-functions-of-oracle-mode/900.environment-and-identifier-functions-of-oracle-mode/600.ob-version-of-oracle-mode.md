# OB_VERSION

## Syntax

```sql
OB_VERSION()
```

## Note

`OB_VERSION()` returns the version of OceanBase Database in the utf8mb4 character set.

## Examples

```sql
obclient> SELECT OB_VERSION() FROM DUAL;
+--------------+
| OB_VERSION() |
+--------------+
| 4.0.0.0      |
+--------------+
1 row in set
```
