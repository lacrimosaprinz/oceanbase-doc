# OLD_CURRENT_SCHEMA

The `OLD_CURRENT_SCHEMA` function returns a session value from `sys_context` ('`userenv`', '`current_schema`').

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This topic applies only to OceanBase Database Enterprise Edition. OceanBase Database Community Edition provides only the MySQL mode. </p>
  </main>

## Syntax

```sql
DBMS_UTILITY.OLD_CURRENT_SCHEMA
RETURN VARCHAR2;
```
