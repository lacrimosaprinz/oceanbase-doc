# CURRENT_INSTANCE


The `CURRENT_INSTANCE` function returns the ID of the currently connected instance. When the connected instance is stopped, `NULL` is returned.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This topic applies only to OceanBase Database Enterprise Edition. OceanBase Database Community Edition provides only the MySQL mode. </p>
  </main>

## Syntax

```sql
DBMS_UTILITY.CURRENT_INSTANCE
RETURN NUMBER;
```
