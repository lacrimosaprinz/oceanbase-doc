# DEBUG_OFF

The `DEBUG_OFF` stored procedure notifies the target session that debugging must end. Calling `DEBUG_OFF` before ending the session is unnecessary.

  <main id="notice" >
    <h4>Applicability</h4>
    <p>This topic applies only to OceanBase Database Enterprise Edition. OceanBase Database Community Edition provides only the MySQL mode. </p>
  </main>

## Syntax

```sql
DBMS_DEBUG.DEBUG_OFF;
```
