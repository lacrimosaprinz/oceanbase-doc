# backup_data_file_size

`backup_data_file_size` specifies the size of the backup data files. You must specify the unit. By default, MB is used.

| **Attribute** | **Description** |
|------------------|-------------|
| Type | Capacity |
| Default value | 4G |
| Value range | \[512M, 4G\] |
| Effective upon OBServer node restart | No |
