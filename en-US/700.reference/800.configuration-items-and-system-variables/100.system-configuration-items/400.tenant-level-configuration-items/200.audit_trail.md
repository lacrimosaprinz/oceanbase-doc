audit_trail
================================

`audit_trail` specifies whether to enable database audit.


| Attribute | Description |
|------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Type | String |
| Default value | NONE |
| Value range | <li> NONE   <li> OS   <li> DB   <li> DB, EXTENDED   <li> DB_EXTENDED |
| Effective upon OBServer node restart | No |
