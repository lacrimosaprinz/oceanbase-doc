# auto_increment_offset

`auto_increment_offset` specifies the starting value for an auto-increment column.

| **Attribute** | **Description** |
|--------|------------------------------------------------------------------------------------------------------------|
| Type | UINT |
| Default value | 1 |
| Value range | \[1, 65535\] |
| Applicable scope | <li> Global   <li> Session |
