# 分区键

分区键是一个或多个列的集合，用于确定分区表中的每一行所在的分区。每一行都明确地分配给一个分区。

例如，对于一个用户表，可以指定 `user_id` 列作为 Range 分区的键。数据库根据此列中的用户号是否在指定范围内，将行分配给分区。

OceanBase 数据库使用分区键自动将插入、更新和删除操作定向到相应的分区。
