# tenant dump info

日志中搜索 "dump tenant info" 关键字，可看到租户的规格、线程、队列、请求统计等信息。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>在 OceanBase 数据库 V4.x 版本，该条日志每个租户每 30 s 打印一次;对于 3.x及以下版本，每 10 s 打印一次。</p>
  </main>

本文日志以 OceanBase 数据库 V4.x 版本为例，其余版本只包含以下日志示例中的部分字段。
