# 删除 DBLink

<main id="notice" >
<h4>功能适用性</h4>
<p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版暂不支持 DBLink 功能。</p>
</main>

您可以删除不需要使用的 DBLink。

## 操作步骤

删除 DBLink 的语法如下：

```sql
DROP DATABASE LINK [IF EXISTS] dblink_name;
```

语句使用说明：

* 执行该语句的用户需要具备 `DROP DATABASE LINK` 权限。为用户授权的相关操作请参见 [修改用户权限](../../../../600.manage/500.security-and-permissions/300.access-control/200.user-and-permission/200.permission-of-mysql-mode/500.modify-user-permissions-of-mysql-mode.md)。

* `IF EXISTS`：如果指定 `IF EXISTS`，即使待删除的 DBLink 不存在，也不会报错；如果不指定且待删除的 DBLink 不存在，则系统会报错。  

* `dblink_name` 表示待删除的 DBLink 的名称。

示例如下：

```sql
DROP DATABASE LINK IF EXISTS ob_dblink;
```
