# SET ENCRYPTION

## 描述

该语句用于设置备份密码。

该语句仅在 Session 生效。该密码在发起备份后会被清除，即如果要配置备份密码，每次发起备份前均需要重新配置。

## 语法

```sql
SET ENCRYPTION ON IDENTIFIED BY 'password' ONLY;
```

## 参数解释

|    参数    |    描述    |
|----------|----------|
| password | 备份的加密密码。 |

## 示例

设置备份密码。

```sql
obclient> SET ENCRYPTION ON IDENTIFIED BY  '***1***' ONLY;
```
