# 字面量概述

字面量（Literal）用于表达一个固定值。

许多函数和 SQL 语句都需要指定字面量，字面量也可以作为表达式和条件的一部分。

OceanBase 数据库支持如下字面量：

* [字符串文本](../400.literal-of-mysql-mode/200.string-literal-of-mysql-mode.md)

* [数值](../400.literal-of-mysql-mode/300.numeric-value-of-mysql-mode.md)

* [日期时间值](../400.literal-of-mysql-mode/400.date-and-time-of-mysql-mode.md)

* [位值](../400.literal-of-mysql-mode/500.hexadecimal-numeric-of-mysql-mode.md)

* [布尔值](../400.literal-of-mysql-mode/600.boolean-of-mysql-mode.md)

* [NULL 值](../400.literal-of-mysql-mode/700.null-value-of-mysql-mode.md)
