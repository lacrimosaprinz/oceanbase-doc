# 布尔值

布尔值包括常量 `TRUE` 和 `FALSE`。

常量 `TRUE` 和 `FALSE` 的计算结果分别为 1 和 0。常量名可以使用任何字母，不区分大小写。

```sql
obclient> SELECT TRUE, true, FALSE, false;
+------+------+-------+-------+
| TRUE | true | FALSE | false |
+------+------+-------+-------+
|    1 |    1 |     0 |     0 |
+------+------+-------+-------+
1 row in set
```
