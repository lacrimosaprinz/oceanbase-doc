# MONTHNAME

## 声明

```sql
MONTHNAME(date)
```

## 说明

返回指定日期 `date` 的月份全称。

## 示例

```sql
obclient> SELECT MONTHNAME('2008-02-03');
+-------------------------+
| MONTHNAME('2008-02-03') |
+-------------------------+
| February                |
+-------------------------+
1 row in set
```
