# CURDATE

## 声明

```javascript
CURDATE()
```

## 说明

返回当前日期，不含时间部分。

## 示例

```javascript
obclient> SELECT CURDATE();
+------------+
| CURDATE()  |
+------------+
| 2018-05-05 |
+------------+
1 row in set
```
