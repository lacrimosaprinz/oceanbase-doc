# PERIOD_ADD

## 声明

```sql
PERIOD_ADD(P,N)
```

## 说明

向时间段 `P` 添加 `N` 个月。其中 `P` 的格式为 `YYMM` 或 `YYYYMM`。

返回值的格式为 `YYYYMM`。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>参数 <code>P</code> 不能是 <code>DATE</code> 类型，即非日期值。</p>
  </main>

## 示例

```sql
obclient> SELECT PERIOD_ADD(200801,2);
+----------------------+
| PERIOD_ADD(200801,2) |
+----------------------+
|               200803 |
+----------------------+
1 row in set
```
