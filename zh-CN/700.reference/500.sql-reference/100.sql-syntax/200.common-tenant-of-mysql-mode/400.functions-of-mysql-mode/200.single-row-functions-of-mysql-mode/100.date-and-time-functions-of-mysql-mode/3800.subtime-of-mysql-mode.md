# SUBTIME

## 声明

```sql
SUBTIME(expr1,expr2)
```

## 说明

返回时间或时间日期表达式 `expr1` 与时间表达式 `expr2` 的差值，返回值的格式与 `expr1` 保持一致。

## 示例

```sql
obclient> SELECT SUBTIME('2021-07-14 19:12:59.999999','1 1:1:1.000002');
+--------------------------------------------------------+
| SUBTIME('2021-07-14 19:12:59.999999','1 1:1:1.000002') |
+--------------------------------------------------------+
| 2021-07-13 18:11:58.999997                             |
+--------------------------------------------------------+
1 row in set
```
