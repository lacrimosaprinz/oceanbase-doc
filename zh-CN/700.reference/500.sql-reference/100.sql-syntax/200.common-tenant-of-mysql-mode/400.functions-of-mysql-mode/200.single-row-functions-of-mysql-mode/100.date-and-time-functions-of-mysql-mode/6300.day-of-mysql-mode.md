# DAY

## 声明

```sql
DAY(date)
```

## 说明

返回 `date` 中月份的日期，范围为 1 到 31。`DAY()` 是 `DAYOFMONTH()` 的同义词。

## 示例

```sql
obclient> SELECT DAY('2008-12-31');
+--------------------------+
| DAY('2008-12-31') |
+--------------------------+
|                       31 |
+--------------------------+
```
