# LEFT

## 声明

```sql
LEFT(str,len)
```

## 说明

返回字符串 `str` 从左侧起的 `len` 个字符，如果 `str` 或 `len` 为 `NULL`，则返回 `NULL`。

该函数是多字节安全的。

## 示例

```sql
obclient> SELECT LEFT('oceanbase', 5);
+----------------------+
| LEFT('oceanbase', 5) |
+----------------------+
| ocean                |
+----------------------+
1 row in set
```
