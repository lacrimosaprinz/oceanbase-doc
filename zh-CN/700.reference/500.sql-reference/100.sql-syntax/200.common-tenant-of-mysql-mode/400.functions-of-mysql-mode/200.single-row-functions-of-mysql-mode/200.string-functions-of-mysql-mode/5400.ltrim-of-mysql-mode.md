# LTRIM

## 声明

```sql
LTRIM(str)
```

## 说明

删除字符串左侧的空格。

## 示例

```sql
obclient> SELECT LTRIM('    obclient    ') AS ltrim;
+--------------+
| ltrim        |
+--------------+
| obclient     |
+--------------+
1 row in set
```
