# RTRIM

## 声明

```sql
RTRIM(str)
```

## 说明

删除字符串 `str` 右侧的空格。

## 示例

```sql
obclient> SELECT RTRIM('    obclient    ') AS rtrim;
+--------------+
| rtrim        |
+--------------+
|     obclient |
+--------------+
1 row in set
```
