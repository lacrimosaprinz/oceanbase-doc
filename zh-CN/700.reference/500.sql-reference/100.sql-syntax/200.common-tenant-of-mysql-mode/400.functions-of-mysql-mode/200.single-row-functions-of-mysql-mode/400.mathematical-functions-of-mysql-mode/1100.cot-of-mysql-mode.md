# COT

## 声明

```sql
COT(X)
```

## 说明

返回 `X` 的余切值。

## 示例

```sql
obclient> SELECT COT(-10), COT(10), COT(-0.5), COT(0.5);
+---------------------+--------------------+--------------------+-------------------+
| COT(-10)            | COT(10)            | COT(-0.5)          | COT(0.5)          |
+---------------------+--------------------+--------------------+-------------------+
| -1.5423510453569202 | 1.5423510453569202 | -1.830487721712452 | 1.830487721712452 |
+---------------------+--------------------+--------------------+-------------------+
1 row in set
```
