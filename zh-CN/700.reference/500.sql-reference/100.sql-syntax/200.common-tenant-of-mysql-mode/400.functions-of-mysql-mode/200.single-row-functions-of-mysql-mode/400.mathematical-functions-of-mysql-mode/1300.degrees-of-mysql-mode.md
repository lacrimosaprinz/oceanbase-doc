# DEGREES

## 语法声明

```sql
DEGREES(x)
```

## 说明

`DEGREES()` 函数将参数 `x` 从弧度转换为度数并返回。

## 示例

如下示例分别将 `PI()` 的值和 `PI()/2` 的值转为对应的度数。

```sql
obclient> SELECT DEGREES(PI());
+---------------+
| DEGREES(PI()) |
+---------------+
|           180 |
+---------------+
1 row in set

obclient> SELECT DEGREES(PI()/2);
+-------------------+
| DEGREES(PI()/2) |
+-------------------+
|                90 |
+-------------------+
1 row in set
```
