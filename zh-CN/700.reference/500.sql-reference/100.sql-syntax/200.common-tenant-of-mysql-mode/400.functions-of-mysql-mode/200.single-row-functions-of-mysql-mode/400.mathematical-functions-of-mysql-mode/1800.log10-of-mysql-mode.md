# LOG10

## 声明

```sql
LOG10(X)
```

## 说明

返回以 10 为底的 `X` 的对数。`X` 是一个正实数，即你想要计算其以 10 为底的对数的数值。

## 示例

```sql
obclient> SELECT LOG10(1000);
+-------------+
| LOG10(1000) |
+-------------+
|           3 |
+-------------+
1 row in set

obclient> SELECT LOG10(-1000);
+--------------+
| LOG10(-1000) |
+--------------+
|         NULL |
+--------------+
1 row in set, 1 warning
```
