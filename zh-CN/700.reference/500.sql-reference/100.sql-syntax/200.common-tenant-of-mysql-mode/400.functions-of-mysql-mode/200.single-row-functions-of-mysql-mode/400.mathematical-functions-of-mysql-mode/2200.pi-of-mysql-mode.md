# PI

## 声明

```sql
PI()
```

## 说明

返回 `π` 的值（圆周率）。

## 示例

```sql
obclient> SELECT PI();
+----------+
| PI()     |
+----------+
| 3.141593 |
+----------+
1 row in set

obclient> SELECT ROUND(2*PI(),2);
+----------------+
| ROUND(2*PI(),2)|
+----------------+
| 6.28           |
+----------------+
1 row in set
```
