# TRUNCATE

## 声明

```sql
TRUNCATE(X,D)
```

## 说明

返回 `X` 保留 `D` 位小数的结果。如果 `D` 为 `0`，则返回 `X` 的整数部分。如果 `D` 为负数，则使 `X` 的小数点左侧的 `D` 位数变成 `0`。

## 示例

```sql
obclient> SELECT TRUNCATE(3.27,1);
+------------------+
| TRUNCATE(3.27,1) |
+------------------+
|              3.2 |
+------------------+
1 row in set

obclient> SELECT TRUNCATE(3.27,0);
+------------------+
| TRUNCATE(3.27,0) |
+------------------+
|                3 |
+------------------+
1 row in set

obclient> SELECT TRUNCATE(3.27,-1);
+-------------------+
| TRUNCATE(3.27,-1) |
+-------------------+
|                 0 |
+-------------------+
1 row in set
```
