# ATAN2

## 声明

```sql
ATAN2(X,Y)
```

## 说明

返回 `X` 与 `Y` 的反正切。它类似于计算 `Y/X` 的反正切，两个参数的符号用于确定结果所在的象限。

该函数也可写作 `ATAN(X,Y)`，请参见 [ATAN](../400.mathematical-functions-of-mysql-mode/400.atan-of-mysql-mode.md)。

## 示例

```sql
obclient> SELECT ATAN(PI(),0);
+--------------------+
| ATAN(PI(),0)       |
+--------------------+
| 1.5707963267948966 |
+--------------------+
1 row in set
```
