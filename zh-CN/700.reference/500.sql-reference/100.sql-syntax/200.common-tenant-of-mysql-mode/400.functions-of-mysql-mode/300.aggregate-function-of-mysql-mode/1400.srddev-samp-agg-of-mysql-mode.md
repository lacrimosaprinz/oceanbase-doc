# STDDEV_SAMP

## 声明

```sql
STDDEV_SAMP(expr)
```

## 说明

返回 `expr` 的样本标准差，该函数返回 `VAR_SAMP()` 函数结果的平方根。有关 `VAR_SAMP()` 函数的详细信息，请参见 [VAR_SAMP](../300.aggregate-function-of-mysql-mode/1700.var-samp-agg-of-mysql-mode.md)。

如果没有匹配到行，则该函数返回 `NULL`。
