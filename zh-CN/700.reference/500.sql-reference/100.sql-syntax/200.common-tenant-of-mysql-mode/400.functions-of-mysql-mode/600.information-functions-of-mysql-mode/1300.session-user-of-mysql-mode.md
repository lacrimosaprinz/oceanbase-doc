# SESSION_USER

## 声明

```sql
SESSION_USER()
```

## 说明

返回当前的用户名与主机名，返回值使用的字符集为 utf8mb4。

该函数是 `USER()` 的同义词。

## 示例

```sql
obclient> SELECT SESSION_USER();
+---------------------+
| SESSION_USER()      |
+---------------------+
| root@100.0.0.1 |
+---------------------+
1 row in set
```
