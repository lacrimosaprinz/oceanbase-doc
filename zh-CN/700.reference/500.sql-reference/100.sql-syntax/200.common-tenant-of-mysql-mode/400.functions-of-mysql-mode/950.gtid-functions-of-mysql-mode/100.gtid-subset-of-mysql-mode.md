# GTID_SUBSET()

## 描述

该函数用于验证是否一个 GTID 集合是另一个 GTID 集合的子集，如果是则返回 `true`；否则返回 `false`。

## 语法

```sql
GTID_SUBSET(set1,set2)
```

  <h4>说明</h4>
