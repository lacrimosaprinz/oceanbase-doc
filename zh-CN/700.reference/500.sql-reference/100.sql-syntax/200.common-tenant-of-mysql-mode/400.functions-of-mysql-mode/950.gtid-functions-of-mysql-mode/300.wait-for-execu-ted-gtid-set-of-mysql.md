# WAIT_FOR_EXECUTED_GTID_SET()

## 描述

该函数用来等待指定的 GTID 在副本上被执行。

## 语法

```sql
WAIT_FOR_EXECUTED_GTID_SET(gtid_set[, timeout])
```
