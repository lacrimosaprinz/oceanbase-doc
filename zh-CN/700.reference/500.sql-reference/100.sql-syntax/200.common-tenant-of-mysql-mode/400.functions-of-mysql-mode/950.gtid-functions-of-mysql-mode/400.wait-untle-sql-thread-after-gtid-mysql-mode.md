# WAIT_UNTIL_SQL_THREAD_AFTER_GTIDS()

## 描述

该函数用于等待直到 SQL 线程执行了指定的 GTID 集合中的所有事务。

## 语法

```sql
WAIT_UNTIL_SQL_THREAD_AFTER_GTIDS(gtid_set[, timeout][,channel])
```

  <h4>说明</h4>
  <p>仅支持该关键字语法，功能不生效。</p>
