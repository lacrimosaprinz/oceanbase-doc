# ALTER LOGFILE GROUP

## 描述

该语句用来修改日志文件组。

## 语法

```sql
ALTER LOGFILE GROUP logfile_group
    ADD UNDOFILE 'file_name'
    [INITIAL_SIZE [=] size]
    [WAIT]
    ENGINE [=] engine_name
```
