# DROP EVENT

该语句用于删除事件。

## 权限要求

执行 `DROP EVENT` 语句创建用户需要当前用户拥有全局的 `DROP` 权限。有关 OceanBase 数据库权限的详细介绍，请参见 [MySQL 模式下的权限分类](../../../../../600.manage/500.security-and-permissions/300.access-control/200.user-and-permission/200.permission-of-mysql-mode/100.permission-classification-of-mysql.md)。

## 语法

```sql
DROP EVENT [IF EXISTS] event_name
```

## 参数解释

| 参数   | 描述     |
|-------|-----------|
|event_name|指定待删除事件的名称。|

## 示例

```shell
obclient> DROP EVENT myenent;
```
