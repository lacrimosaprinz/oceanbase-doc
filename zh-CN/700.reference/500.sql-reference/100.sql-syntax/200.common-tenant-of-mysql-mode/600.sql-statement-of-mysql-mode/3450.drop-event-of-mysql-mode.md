# DROP EVENT

该语句用于删除事件。

## 语法

```sql
DROP EVENT [IF EXISTS] event_name
```

## 参数解释

| 参数   | 描述     |
|-------|-----------|
|event_name|指定待删除事件的名称。|

## 示例

```shell
obclient> DROP EVENT myevent;
```
