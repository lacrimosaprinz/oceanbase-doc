# EXECUTE

## 描述

该语句用于执行准备的语句。

## 语法

```sql
EXECUTE stmt_name
[USING @var_name [, @var_name] ...]
```

## 参数解释

|    参数     |           描述            |
|-----------|-------------------------|
| stmt_name | 准备语句的名称。                |
| @var_name | 按顺序填入 `stmt_name` 中的变量。 |

## 示例

执行准备的语句。

```sql
obclient> PREPARE stmt1 FROM 'SELECT SQRT(POW(?,2) + POW(?,2)) AS hypotenuse';
Query OK, 0 rows affected

obclient> SET @a = 3;
Query OK, 0 rows affected

obclient> SET @b = 4;
Query OK, 0 rows affected

obclient> EXECUTE stmt1 USING @a, @b;
Query OK, 0 rows affected
```
