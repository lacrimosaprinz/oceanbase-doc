# PURGE RECYCLEBIN

## 描述

该语句用于清空回收站。

  <main id="notice" type='notice'>
    <h4>注意</h4>
    <p><code>PURGE RECYCLEBIN</code> 会清除回收站中的所有数据库对象，只有 root 用户才能够执行这条语句，操作时请谨慎。</p>
  </main>

## 语法

```sql
 PURGE RECYCLEBIN;
```

## 示例

清空回收站。

```sql
obclient> PURGE RECYCLEBIN;
Query OK, 0 rows affected
```
