# 字符串到日期的转换规则

将字符串值转换为日期值时需要遵循一定的转换规则。

转换规则如下：

* 如果指定了数值格式模型元素的所有数值（包括前导零点），则可以从日期字符串中省略格式字符串中包含的标点符号。例如，为两位数格式化元素（如 `MM`、`DD` 和 `YY`）指定 02 而不是 2。

* 您可以从日期字符串中省略在格式字符串末尾找到的时间字段。

* 您可以在日期字符串中使用任何非字母数值字符来匹配格式字符串中的标点符号。
