# ATAN

## 描述

该函数用于求反正切值。反正切值是指正切值为`number` 的角度，返回的角度以弧度表示，弧度值范围为 (-π/2，π/2)。

## 语法

```sql
ATAN(number)
```

## 参数解释

`number` 是一个数值数据类型或可以隐式转换为数值数据类型的参数。

## 返回类型

返回 `NUMBER` 数据类型。

## 示例

返回 `0.5` 的反正切值。

```sql
obclient> SELECT ATAN(0.5) FROM DUAL;
+-------------------------------------------+
| ATAN(0.5)                                 |
+-------------------------------------------+
| .4636476090008061162142562314612144020285 |
+-------------------------------------------+
1 row in set
```
