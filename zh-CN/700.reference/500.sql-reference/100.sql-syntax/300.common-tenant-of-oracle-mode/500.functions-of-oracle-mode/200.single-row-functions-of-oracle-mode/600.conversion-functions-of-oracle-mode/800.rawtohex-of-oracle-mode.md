# RAWTOHEX

## 描述

该函数将一个二进制格式的字符串转换为一个相应的十六进制表示的字符串。

## 语法

```sql
RAWTOHEX (raw)
```

## 参数解释

| 参数  |    说明    |
|-----|----------|
| raw | 二进制的字符串。 |

## 返回类型

十六进制表示的字符串。

## 示例

将二进制 `AB` 转化为对应的十六进制字符串。

```sql
obclient> SELECT RAWTOHEX('AB')  FROM DUAL;
+----------------+
| RAWTOHEX('AB') |
+----------------+
| 4142           |
+----------------+
1 row in set
```
