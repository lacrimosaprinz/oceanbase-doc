# GROUPING

## 描述

该函数将 `GROUP BY` 子句扩展（如 `ROLLUP`）生成的超聚集行中的 `NULL` 和常规行中的 `NULL` 区分开来。该函数把超聚集行中的 `NULL` 返回 `1`，常规行中的值返回 `0`。

## 语法

```sql
GROUPING(expr)
```

## 参数解释

参数 `expr` 是 `GROUP BY` 子句中的一个表达式。

## 返回类型

返回 `NUMBER` 数据类型。

## 示例

创建表 `tbl1`，并插入数据。查询以列 `col2` 和列 `col3` 分组及 `ROLLUP` 子句生成的汇总行和正常记录。
