# ROW_NUMBER

## 描述

该函数是一个分析函数，作用是为每一条行（分组）记录返回一个唯一的序号，该序号按照 `order_by_clause` 中指定列进行的排序，从 `1` 开始。当遇到相同的数据时，排名按照记录集中记录的顺序依次递增，对于不同数据进行依次排名。

## 语法

```sql
ROW_NUMBER( ) OVER ([ query_partition_clause ] order_by_clause)
```

## 参数解释

使用 `OVER` 子句定义窗口进行计算。详细信息请参见 [分析函数说明](../400.analysis-functions-of-oracle-mode/100.window-function-description-of-oracle-mode.md)。

## 返回类型

返回数值类型数据。
