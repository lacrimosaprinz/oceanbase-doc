# CONNECTION_ID

## 声明

```sql
OB_TRANSACTION_ID()
```

## 说明

* 当前会话处于活跃事务中，则返回所处的事务 ID 整数值，大于 0。
* 当前会话如果未处于活跃事务中，返回 0。

## 示例

1. 开启事务。

    ```shell
    obclient [SYS]> START TRANSACTION;
    ```

2. 查询当前处于活跃事务中的事务 ID。

    ```shell
    obclient [SYS]> SELECT OB_TRANSACTION_ID() FROM DUAL;
    ```

    返回结果如下：

    ```shell
    +---------------------+
    | OB_TRANSACTION_ID() |
    +---------------------+
    |            11003721 |
    +---------------------+
    1 row in set
    ```

3. 提交事务。

    ```shell
    obclient [SYS]> COMMIT;
    ```

4. 查询当前未处于活跃事务中的事务 ID。

    ```shell
    obclient [SYS]> SELECT OB_TRANSACTION_ID() FROM DUAL;
    ```

    返回结果如下：

    ```shell
    +---------------------+
    | OB_TRANSACTION_ID() |
    +---------------------+
    |                   0 |
    +---------------------+
    1 row in set
    ```
