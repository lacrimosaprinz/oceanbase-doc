# 复合条件

复合条件（Compound Conditions）是多个其他条件的组合。

复合条件的语法如下：

```sql
{ (condition) | NOT condition | condition { AND | OR } condition }
```

关于条件 `NOT`、`AND` 和 `OR` 条件的详细信息，请参见 [逻辑条件](../700.condition-of-oracle-mode/500.logical-conditions-of-oracle-mode.md)。
