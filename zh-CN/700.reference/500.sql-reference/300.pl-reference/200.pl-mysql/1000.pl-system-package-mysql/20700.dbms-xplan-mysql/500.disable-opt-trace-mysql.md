# DISABLE_OPT_TRACE

`DISABLE_OPT_TRACE` 过程用于关闭当前 Session 的优化器全链路追踪功能。

## 语法

```sql
DBMS_XPLAN.DISABLE_OPT_TRACE;
```

## 示例

关闭当前 Session 的优化器追踪功能。

```shell
obclient> CALL DBMS_XPLAN.DISABLE_OPT_TRACE();
Query OK, 0 rows affected 
```
