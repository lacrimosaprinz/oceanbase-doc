# PURGE_LOG

`PURGE_LOG` 指定基表通过刷新相关物化视图清除物化视图日志数据。

<main id="notice" >
  <h4>功能适用性</h4>
  <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
</main>

## 语法

```sql
PROCEDURE purge_log(
    IN     master_name            VARCHAR(65535));
```

## 使用限制

<main id="notice" >
  <h4>功能适用性</h4>
  <p>该语法中暂不支持参数 `num` 和 `flag`。</p>
</main>

## 参数解释

| **参数**           | **解释**                                                                                                                                              |
|------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| master      | 需要清除数据的物化视图日志的主表名称
格式为 `schema.object or object`。                                                                         |
