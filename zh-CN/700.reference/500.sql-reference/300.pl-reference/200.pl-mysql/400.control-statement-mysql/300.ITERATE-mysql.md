# ITERATE 


`ITERATE` 语句用于再次启动循环语句。

`ITERATE` 语句的语法如下：

```sql
ITERATE label
```

`ITERATE` 只能出现在 `LOOP`、`REPEAT` 和 `WHILE` 语句中。
