# DO 


`DO` 语句用于执行 SQL 表达式，但不返回任何结果。

`DO` 语句的语法如下：

```sql
DO expr [, expr ...]
```


在一些场景中，`DO` 语句等价于 `SELECT expr`，以实现快速响应而不用返回结果。示例如下：

```sql
obclient> SELECT SLEEP(5);
+----------+
| SLEEP(5) |
+----------+
|        0 |
+----------+
1 row in set (5.001 sec)

obclient> DO SLEEP(5);
Query OK, 0 rows affected (5.011 sec)
```
