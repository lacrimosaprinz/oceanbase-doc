# INITIALIZE 

`INITIALIZE` 存储过程用于使用种子值进行包的初始化。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法 

```sql
DBMS_RANDOM.INITIALIZE (
val  IN  BINARY_INTEGER);
```



## 参数解释 



| **参数** |    **解释**    |
|--------|--------------|
| val    | 用于生成随机码的种子数。 |



## 使用说明 


此程序不执行任何功能，尽管当前支持此程序，但已过期，不建议使用。
