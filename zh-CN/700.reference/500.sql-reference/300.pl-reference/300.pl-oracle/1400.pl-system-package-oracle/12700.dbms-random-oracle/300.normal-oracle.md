# NORMAL 

`NORMAL` 函数用于返回标准正态分布的随机码。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法 

```sql
DBMS_RANDOM.NORMAL
RETURN NUMBER;
```



## 返回值 



| **参数** |  **描述**  |
|--------|----------|
| number | 返回一个随机码。 |
