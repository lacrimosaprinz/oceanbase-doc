# CLEAR_IDENTIFIER 

`CLEAR_IDENTIFIER` 过程将当前会话的标识设置为 `NULL`。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法 

```sql
DBMS_SESSION.CLEAR_IDENTIFIER;
```


## 使用说明 

`CLEAR_IDENTIFIER` 过程可由 `PUBLIC` 执行。

## 示例 

```sql
BEGIN
  DBMS_SESSION.CLEAR_IDENTIFIER;
END;
```
