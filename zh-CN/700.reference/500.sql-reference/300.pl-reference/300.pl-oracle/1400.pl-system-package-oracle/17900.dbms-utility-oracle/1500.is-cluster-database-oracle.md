# IS_CLUSTER_DATABASE 

`IS_CLUSTER_DATABASE` 函数用于确定数据库是否在集群数据库模式下运行。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法 

```sql
DBMS_UTILITY.IS_CLUSTER_DATABASE 
RETURN BOOLEAN;
```



## 返回值 

如果此实例以集群数据库模式启动，则此函数返回 `TRUE`；否则返回 `FALSE`。
