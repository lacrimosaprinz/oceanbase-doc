# FORMAT_ERROR_BACKTRACE 

`FORMAT_ERROR_BACKTRACE` 函数用于格式化引发异常的调用堆栈的回溯。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法 

```sql
DBMS_UTILITY.FORMAT_ERROR_BACKTRACE 
RETURN VARCHAR2;
```



## 返回值 

回溯字符串。如果当前没有处理错误，则返回 `NULL`。
