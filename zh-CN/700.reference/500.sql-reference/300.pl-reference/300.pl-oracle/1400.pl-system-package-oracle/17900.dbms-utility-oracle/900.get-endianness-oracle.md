# GET_ENDIANNESS 

`GET_ENDIANNESS` 函数用于获取数据库平台的字节存储顺序。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法 

```sql
DBMS_UTILITY.GET_ENDIANNESS
RETURN NUMBER;
```



## 返回值 

一个 `NUMBER` 类型的值，表示数据库平台的字节存储顺序。使用 1 表示大端或 2 表示小端。
