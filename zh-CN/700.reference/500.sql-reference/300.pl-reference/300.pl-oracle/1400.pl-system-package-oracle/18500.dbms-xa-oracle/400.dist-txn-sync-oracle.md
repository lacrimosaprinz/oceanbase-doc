# DIST_TXN_SYNC 

当 Real Application Clusters（ RAC）实例之一发生故障时，可以使用此子程序同步不确定事务。


  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法 

```sql
DBMS_XA.DIST_TXN_SYNC; 
```


  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>OceanBase 数据库支持此过程仅做兼容使用，并无实质操作。</p>
  </main>
