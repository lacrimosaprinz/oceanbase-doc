# ROLLUP_UNIT

`ROLLUP_UNIT` 用于计算某次执行中某个单元的执行时间。

 <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法

```sql
DBMS_PROFILER.rollup_unit
  RETURN BINARY_INTEGER; 
```

## 示例

```shell
obclient> call DBMS_PROFILER.rollup_unit();
```
