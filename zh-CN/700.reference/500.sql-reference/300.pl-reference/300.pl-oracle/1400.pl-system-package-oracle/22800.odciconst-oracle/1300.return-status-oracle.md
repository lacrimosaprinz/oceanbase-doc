# Return Status 

`Return Status` 常量及其描述。


|    常量名称     |  常量类型   | 默认值 |                   描述                   |
|-------------|---------|-----|----------------------------------------|
| Success     | INTEGER | 0   | 表示操作成功                                 |
| Error       | INTEGER | 1   | 表示发生错误                                 |
| Warning     | INTEGER | 2   | 表示警告                                   |
| ErrContinue | INTEGER | 3   | 表示索引分区中有错误，但继续在下一个分区上工作                |
| Fatal       | INTEGER | 4   | 表示索引的所有字典条目均已清理，并且 `CREATE INDEX` 操作已回滚 |
