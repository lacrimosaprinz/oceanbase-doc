# FREMOVE 

`FREMOVE` 存储过程用于删除指定的文件。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法 

```sql
UTL_FILE.FREMOVE (
location IN VARCHAR2,
filename IN VARCHAR2);
```



## 参数解释 



|    参数    |       解释        |
|----------|-----------------|
| location | 文件所在的路径（大小写敏感）。 |
| filename | 需要被删除的文件名。      |



## 异常情况 

* `ACCESS_DENIED`

  

* `DELETE_FAILED`

  

* `INVALID_FILENAME`

  

* `INVALID_OPERATION`

  

* `INVALID_PATH`

  




## 使用说明 

`FREMOVE` 在删除文件之前不会验证权限。但操作系统会验证文件和目录权限。
