# BIT_COMPLEMENT 

`BIT_COMPLEMENT` 函数对 `RAW r` 中的值执行按位逻辑“补码”，并返回补码结果 `RAW`。结果长度等于输入的 `RAW r `长度。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法 

```sql
UTL_RAW.BIT_COMPLEMENT (
   r IN RAW) 
  RETURN RAW;
```



## 参数解释 

| **参数** |    **解释**     |
|--------|---------------|
| r      | RAW 执行"补码"操作。 |



## 返回值 

| **返回值** |    **描述**      |
|---------|------------------|
| RAW     | `r` 的"补码"。       |
