# ENCRYPT 

`ENCRYPT` 函数使用流密码或块密码加密 RAW 数据，该密码使用用户提供的密钥和可选的 IV（初始化向量）。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法 


```sql
DBMS_CRYPTO.ENCRYPT(
   src IN RAW,
   typ IN PLS_INTEGER,
   key IN RAW,
   iv  IN RAW DEFAULT NULL)
 RETURN RAW;
```



## 参数解释 



| **参数** |         **解释**         |
|--------|------------------------|
| src    | 要加密的 RAW 数据。           |
| typ    | 所使用的密码流和密码块链接算法。       |
| key    | 用于加密的密钥。               |
| iv     | 密码块的可选初始化向量。默认值为 NULL。 |



## 使用说明 

* 密码块可以使用链接和填充算法进行修改。链接和填充算法被添加到密码块中，以生成密码组合。密码块链接（CBC）是最常用的链接类型。关于此系统包所支持的密码块链接和填充算法，请参见 [DBMS_CRYPTO 概述](../3800.dbms-crypto-oracle/100.dbms-crypto-overview-oracle.md)。

  

* 要加密 VARCHAR2 数据，应首先将其转换为 AL32UTF8 字符集。
