# GET_VALUES 

`GET_VALUES` 函数用于返回当前栈帧的所有变量信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法 

```sql
DBMS_DEBUG.GET_VALUES(
scalar_values OUT VARCHAR2)
RETURN binary_integer;
```



## 参数解释 



|    **参数**    | **解释** |
|--------------|--------|
| scalar_value | 值。     |
