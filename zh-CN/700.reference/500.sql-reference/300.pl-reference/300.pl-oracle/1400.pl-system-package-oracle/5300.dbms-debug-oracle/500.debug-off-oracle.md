# DEBUG_OFF 

`DEBUG_OFF` 存储过程用于通知目标会话应该停止调试。在结束会话之前不必调用此函数。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

## 语法 

```sql
DBMS_DEBUG.DEBUG_OFF;
```
