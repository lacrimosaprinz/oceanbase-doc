# oceanbase.CDB_TAB_COLS_V$

## 功能

展示所有租户的表、视图对象的列信息。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始引入。</p>
</main>
