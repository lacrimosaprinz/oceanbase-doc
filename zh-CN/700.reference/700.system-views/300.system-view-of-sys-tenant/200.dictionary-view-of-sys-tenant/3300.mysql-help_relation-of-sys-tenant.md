# mysql.help_relation

## 功能

与其他三个表 `mysql.help_category`、`mysql.help_topic`、`mysql.help_keyword` 合作生成 help 命令。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V1.4 版本开始引入。</p>
</main>

## 相关表/视图

* `mysql.help_category`

* `mysql.help_topic`

* `mysql.help_keyword`

## 字段说明

|    **字段名称**     |   **类型**   | **是否可以为 NULL** | **描述** |
|-----------------|------------|----------------|--------|
| help_topic_id   | bigint(20) | NO             | 主题  ID |
| help_keyword_id | bigint(20) | NO             | 关键词 ID |
