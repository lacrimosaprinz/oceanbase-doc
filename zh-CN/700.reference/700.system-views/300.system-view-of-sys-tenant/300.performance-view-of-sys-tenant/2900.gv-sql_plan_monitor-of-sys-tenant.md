# GV$SQL_PLAN_MONITOR

## 功能

展示所有 OBServer 节点慢查询的 Plan 层面的统计，每个慢查询都会有一条统计信息，同时记录该 Plan 的 Trace 信息。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V1.4 版本开始引入。</p>
</main>
