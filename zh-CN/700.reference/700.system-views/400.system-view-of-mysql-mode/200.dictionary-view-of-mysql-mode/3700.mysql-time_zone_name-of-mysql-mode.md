# mysql.time_zone_name

## 功能

展示 timezone 名称信息（SYSTEM_VIEW）。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V1.4 版本开始引入。</p>
</main>

## 字段说明

|     字段名称     |     类型      | 是否可以为 NULL |  描述   |
|--------------|-------------|------------|-------|
| Name         | varchar(64) | NO         | 时区名称  |
| Time_zone_id | bigint(20)  | NO         | 时区 ID |
