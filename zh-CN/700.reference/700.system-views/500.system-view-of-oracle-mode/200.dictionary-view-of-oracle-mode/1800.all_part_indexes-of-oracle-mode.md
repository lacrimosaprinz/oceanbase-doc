ALL_PART_INDEXES
=====================================

功能
-----------

查看当前用户可访问的分区索引的分区信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

相关视图
-------------

* DBA_PART_INDEXES

* USER_PART_INDEXES

字段说明

| OWNER                     | VARCHAR2(128)  | NO             | 分区索引的拥有者                                                                                |
