DICT
=========================

功能
-----------

展示数据字典表和视图的描述。与 `DICTIONARY` 视图相同。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

字段说明
-------------

|  **字段名称**  |     **类型**     | **是否可以为 NULL** | **描述**  |
|------------|----------------|----------------|---------|
| TABLE_NAME | VARCHAR2(128)  | NO             | 对象的名称   |
| COMMENTS   | VARCHAR2(4000) | NO             | 对象的文本注释 |
