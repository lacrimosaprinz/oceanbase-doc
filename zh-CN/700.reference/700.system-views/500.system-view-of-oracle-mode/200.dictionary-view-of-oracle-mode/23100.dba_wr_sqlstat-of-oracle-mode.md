# DBA_WR_SQLSTAT

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该视图从 V4.2.2 版本开始引入。</p>
</main>

## 功能描述

视图 `DBA_WR_SQLSTAT` 用于展示本租户执行过的 SQL 的基本性能统计数据。其中，含 `_DELTA` 的列表示从上次采集 WR 快照到当前时间为止统计值的增量。
