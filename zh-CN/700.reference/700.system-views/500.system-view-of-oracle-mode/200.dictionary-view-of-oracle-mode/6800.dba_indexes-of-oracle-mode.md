DBA_INDEXES
================================

功能
-----------

`DBA_INDEXES` 用于查看数据库所有表的索引信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

相关视图
-------------

* `USER_INDEXES`

* `ALL_INDEXES`
