V$TIMEZONE_NAMES 
=====================================

**功能** 
---------------------------

显示有效的时区名称。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

**字段说明** 
-----------------------------



| **字段名称** |    **类型**     | **是否可以为 NULL** |     **描述**     |
|----------|---------------|----------------|----------------|
| TZNAME   | varchar(64)   | NO             | 时区区域           |
| TZABBREV | varchar(8)    | NO             | 相对时间缩写（例如，PDT） |
| CON_ID   | decimal(38,0) | NO             | 租户 ID          |
