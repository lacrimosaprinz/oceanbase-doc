backup_dest 
================================

backup_dest 用于设置基线备份和日志归档的路径。


<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>  从 V4.0.0 版本开始弃用。  </p>
</main>

|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 字符串    |
| 默认值              | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 节点生效 | 否      |


<main id="notice" type='notice'>
  <h4>注意</h4>
  <p>  备份进行过程中不可修改 <code>backup_dest</code> 配置项。  </p>
</main>
