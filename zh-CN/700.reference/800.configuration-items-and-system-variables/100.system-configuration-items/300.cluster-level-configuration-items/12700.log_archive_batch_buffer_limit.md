log_archive_batch_buffer_limit 
===================================================

当物理备份开启时，log_archive_batch_buffer_limit 用于控制日志归档单机的最大内存使用量。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>



|        属性        |    描述     |
|------------------|-----------|
| 参数类型             | 容量类型      |
| 默认值              | 1G        |
| 取值范围             | \[2M, +∞) |
| 是否重启 OBServer 节点生效 | 否         |




**示例**

```sql
obclient> ALTER SYSTEM SET log_archive_batch_buffer_limit = 1GB;
```
