log_archive_checkpoint_interval 
====================================================

`log_archive_checkpoint_interval` 用于设置冷数据的日志归档检查点的时间间隔。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 时间类型       |
| 默认值              | 120s       |
| 取值范围             | \[5s, 1h\] |
| 是否重启 OBServer 节点生效 | 否          |
