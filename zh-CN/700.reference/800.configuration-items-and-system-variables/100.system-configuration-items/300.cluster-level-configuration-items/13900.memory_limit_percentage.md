# memory_limit_percentage

## 功能描述

`memory_limit_percentage` 用于设置系统总可用内存大小占总内存大小的百分比。

## 属性说明

|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 80         |
| 取值范围             | \[10, 95] <main id="notice" type='explain'><h4>说明</h4><p>从 V4.2.1 版本开始默认值由 90 调整为 95。</p></main>|
| 是否重启 OBServer 节点生效 | 否          |
