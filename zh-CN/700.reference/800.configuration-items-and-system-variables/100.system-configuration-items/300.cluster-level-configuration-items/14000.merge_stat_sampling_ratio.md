merge_stat_sampling_ratio 
==============================================

`merge_stat_sampling_ratio` 用于设置合并时候数据列统计信息的采样率。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 100        |
| 取值范围             | \[0, 100\] |
| 是否重启 OBServer 节点生效 | 否          |
