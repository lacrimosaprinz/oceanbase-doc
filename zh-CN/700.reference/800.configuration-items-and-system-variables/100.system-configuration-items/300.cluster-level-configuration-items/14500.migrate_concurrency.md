migrate_concurrency 
========================================

`migrate_concurrency` 用于设置内部数据迁移的最大并发数。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 10        |
| 取值范围             | \[0, 64\] |
| 是否重启 OBServer 节点生效 | 否         |
