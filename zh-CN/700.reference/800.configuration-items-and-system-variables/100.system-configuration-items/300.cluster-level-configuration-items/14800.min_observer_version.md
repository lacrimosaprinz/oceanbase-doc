min_observer_version 
=========================================

`min_observer_version` 用于查看本集群最低的 OBServer 节点应用程序版本号。


|      **属性**      | **描述**  |
|------------------|---------|
| 参数类型             | 字符串     |
| 默认值              | 4.2.1.0 |
| 取值范围             | NULL    |
| 是否重启 OBServer 节点生效 | 否       |
