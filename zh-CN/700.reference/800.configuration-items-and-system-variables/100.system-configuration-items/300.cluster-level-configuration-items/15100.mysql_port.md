# mysql_port 

`mysql_port` 用于设置 SQL 服务协议端口号。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V3.2.4 版本开始默认值由 2880 调整为 2881。</p>
</main>

|      **属性**      |     **描述**      |
|------------------|-----------------|
| 参数类型             | 整型              |
| 默认值              | 2881            |
| 取值范围             | \[1025, 65535\] |
| 是否重启 OBServer 节点生效 | 否               |
