multiblock_read_size 
=========================================

`multiblock_read_size` 用于设置读取数据时 IO 聚合大小。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 容量单位       |
| 默认值              | 128k       |
| 取值范围             | \[0K, 2M\] |
| 是否重启 OBServer 节点生效 | 否          |
