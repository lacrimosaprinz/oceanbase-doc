px_task_size 
=================================

px_task_size 用于设置 SQL 并行查询引擎每个任务处理的数据量大小。配置时需说明单位，默认单位为 MB。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 容量单位      |
| 默认值              | 2M        |
| 取值范围             | \[2M, +∞) |
| 是否重启 OBServer 节点生效 | 否         |
