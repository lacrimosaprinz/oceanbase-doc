# ob_esi_session_timeout

`ob_esi_session_timeout` 用于设置 obesi 进程持有活动会话资源的超时时间。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

| **属性** | **描述** |
| --- | --- |
| 参数类型 | 时间类型 |
| 默认值 | 1m |
| 取值范围 | [10s, 30m] |
| 是否重启 OBServer 节点生效 | 否 |
