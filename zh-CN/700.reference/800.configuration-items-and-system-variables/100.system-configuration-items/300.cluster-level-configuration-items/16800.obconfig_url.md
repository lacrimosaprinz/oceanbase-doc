obconfig_url 
=================================

`obconfig_url` 用于设置 OBConfig 服务的 URL 地址。


|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 字符串类型  |
| 默认值              | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 节点生效 | 否      |
