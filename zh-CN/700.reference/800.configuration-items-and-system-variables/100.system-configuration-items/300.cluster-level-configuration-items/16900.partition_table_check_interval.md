partition_table_check_interval 
===================================================

`partition_table_check_interval` 用于设置 OBServer 从分区表中删除不存在的副本的时间间隔。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>


|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 时间类型         |
| 默认值              | 30m，表示 30 分钟 |
| 取值范围             | \[1m, +∞)    |
| 是否重启 OBServer 节点生效 | 否            |
