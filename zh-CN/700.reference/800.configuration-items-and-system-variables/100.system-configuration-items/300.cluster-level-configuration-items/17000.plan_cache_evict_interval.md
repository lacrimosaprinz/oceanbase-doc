# plan_cache_evict_interval 


plan_cache_evict_interval 用于设置执行计划缓存的淘汰时间间隔。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从 V4.2.0 版本开始默认值由 1s 调整为 5s。</p>
</main

|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 时间类型      |
| 默认值              | 5s       |
| 取值范围             | \[0s, +∞) |
| 是否重启 OBServer 节点生效 | 否         |
