# plan_cache_low_watermark 


plan_cache_low_watermark 用于设置执行计划缓存占用内存的阈值，如果低于该阈值时将停止淘汰。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>该配置项从 V4.2.0 版本开始弃用。</p>
  </main>


|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 容量单位   |
| 默认值              | 1500M  |
| 取值范围             | NULL   |
| 是否重启 OBServer 节点生效 | 否      |
