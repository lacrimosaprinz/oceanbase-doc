rootservice_list 
=====================================

rootservice_list 用于显示 Root Service 及其副本所在的机器列表。


|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 字符串    |
| 默认值              | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 节点生效 | 否      |
