replica_safe_remove_time 
=============================================

`replica_safe_remove_time` 用于设置已删除副本可以被清理的安全保留时间。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 时间类型      |
| 默认值              | 2h        |
| 取值范围             | \[1m, +∞) |
| 是否重启 OBServer 节点生效 | 否         |
