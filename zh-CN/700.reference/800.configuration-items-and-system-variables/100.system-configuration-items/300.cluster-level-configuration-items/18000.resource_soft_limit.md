resource_soft_limit 
========================================

`resource_soft_limit` 用于设置是否开启 Unit 均衡。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>


|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 整型           |
| 默认值              | 50           |
| 取值范围             | \[1, 10000\] |
| 是否重启 OBServer 节点生效 | 否            |


当 `resource_soft_limit` 配置项的取值小于 `100` 时，表示 Unit 均衡开关打开；当 `resource_soft_limit` 配置项的取值大于等于 `100` 时，表示 Unit 均衡开关关闭。
