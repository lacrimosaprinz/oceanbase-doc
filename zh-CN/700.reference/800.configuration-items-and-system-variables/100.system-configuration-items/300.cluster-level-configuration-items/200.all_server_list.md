# all_server_list

`all_server_list` 用于显示群集中的所有服务器地址。

|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 字符串    |
| 默认值              | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 节点生效 | 否      |
