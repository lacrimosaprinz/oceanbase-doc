server_data_copy_in_concurrency 
====================================================

`server_data_copy_in_concurrency` 用于设置单个节点迁入数据的最大并发数。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      |   **描述**    |
|------------------|-------------|
| 参数类型             | 整型          |
| 默认值              | 2           |
| 取值范围             | \[1, 1000\] |
| 是否重启 OBServer 节点生效 | 否           |
