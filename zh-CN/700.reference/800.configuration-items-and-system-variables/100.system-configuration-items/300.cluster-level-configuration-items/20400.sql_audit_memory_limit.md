sql_audit_memory_limit
===========================================

`sql_audit_memory_limit` 用于设置 SQL 审计数据可占用的最大内存。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 容量单位       |
| 默认值              | 3G         |
| 取值范围             | \[64M, +∞) |
| 是否重启 OBServer 节点生效 | 否          |
