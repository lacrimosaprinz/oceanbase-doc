# sql_login_thread_count


`sql_login_thread_count` 用于设置处理 SQL 登录请求的线程数量。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V3.2.4 版本开始引入。</p>
</main>

| **属性** | **描述** |
| --- | --- |
| 参数类型 | 整型 |
| 默认值 | 0，表示使用默认值，在 `mini-mode` 模式下默认值为 2 个线程，在正常模式下默认值为 6 个线程|
| 取值范围 | [0, 32] |
| 是否重启 OBServer 节点生效 | 否 |
