# system_trace_level

system_trace_level 用于设置系统追踪日志的日志打印级别。

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该配置项从 V4.2.0 版本开始弃用。</p>
</main>

|      **属性**      |                                                                               **描述**                                                                                |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 整型                                                                               |
| 默认值              | 1                                                                                |
| 取值范围             | <li> 0：表示 none   <li> 1：表示 standard   <li> 2：表示 debug    |
| 是否重启 OBServer 节点生效 | 否                                                                                |
