sys_bkgd_io_high_percentage 
================================================

`sys_bkgd_io_high_percentage` 用于设置系统后台 IO 最高占用百分比。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 90         |
| 取值范围             | \[1, 100\] |
| 是否重启 OBServer 节点生效 | 否          |
