tenant_groups 
==================================

`tenant_groups` 用于设置负载均衡策略中使用的租户组。


<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 字符串类型  |
| 默认值              | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 节点生效 | 否      |
