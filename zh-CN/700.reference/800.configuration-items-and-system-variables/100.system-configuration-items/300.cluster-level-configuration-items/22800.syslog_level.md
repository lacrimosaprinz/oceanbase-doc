# syslog_level

`syslog_level`  用于设置系统日志级别。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V3.2.4 版本开始默认值由 INFO 调整为 WDIAG。</p>
</main>

| **属性** | **描述** |
| --- | --- |
| 参数类型 | 字符串 |
| 默认值 | WDIAG |
| 取值范围 |<ul><li>  DEBUG </li><li> TRACE</li><li>WDIAG</li><li>EDIAG</li><li>INFO </li><li> WARN</li><li> ERROR</li></ul>|
| 是否重启 OBSever 节点生效 | 否 |
