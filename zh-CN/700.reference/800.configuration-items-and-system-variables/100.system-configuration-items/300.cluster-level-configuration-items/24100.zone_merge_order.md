zone_merge_order 
=====================================

`zone_merge_order` 用于设置 Zone 的轮转合并顺序。不指定时，由系统自动决定。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 字符串类型  |
| 默认值              | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 节点生效 | 否      |
