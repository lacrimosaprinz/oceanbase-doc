trx_try_wait_lock_timeout 
==============================================

`trx_try_wait_lock_timeout` 用于设置语句执行过程行锁的等待时长。


<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V3.1.0 版本开始弃用。</p>
</main>

|      **属性**      |     **描述**     |
|------------------|----------------|
| 参数类型             | 时间类型           |
| 默认值              | 0ms            |
| 取值范围             | \[0ms, 100ms\] |
| 是否重启 OBServer 节点生效 | 否              |
