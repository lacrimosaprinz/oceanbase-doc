# ob_esi_io_concurrency

`ob_esi_io_concurrency` 用于设置 obesi 进程的并发 I/O 数。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

| **属性** | **描述** |
| --- | --- |
| 参数类型 | 整型 |
| 默认值 | 0 |
| 取值范围 | [0, +∞) |
| 是否重启 OBServer 节点生效 | 否 |
