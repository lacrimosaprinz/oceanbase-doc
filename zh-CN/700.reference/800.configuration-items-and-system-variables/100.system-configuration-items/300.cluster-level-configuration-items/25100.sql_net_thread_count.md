# sql_net_thread_count

`sql_net_thread_count` 用于设置 MySQL 集群 IO 线程的个数，即 `global_sql_nio_server` 线程的个数。默认值为 0 表示与配置项 `net_thread_count` 的值相同。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.1.0 版本开始引入。</p>
</main>

| **属性** | **描述** |
| --- | --- |
| 参数类型 | 整型 |
| 默认值 | 0 |
| 取值范围 | \[0, 64] |
| 是否重启 OBSever 节点生效 | 否 |
