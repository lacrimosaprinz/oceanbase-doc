user_iort_up_percentage 
============================================

`user_iort_up_percentage` 用于设置用户磁盘 IO 时延阈值，即用户磁盘 IO 时延超过该阈值后，系统后台 IO 任务将被限流。


<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 整型       |
| 默认值              | 100      |
| 取值范围             | \[0, +∞) |
| 是否重启 OBServer 节点生效 | 否        |
