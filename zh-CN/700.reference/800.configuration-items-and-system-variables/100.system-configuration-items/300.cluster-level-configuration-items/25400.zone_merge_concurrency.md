zone_merge_concurrency 
===========================================

`zone_merge_concurrency` 用于设置在合并时，支持多少个 Zone 并发。当值为 0 时，由系统根据部署情况自动选择最佳并发度。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 整型       |
| 默认值              | 1        |
| 取值范围             | \[0, +∞) |
| 是否重启 OBServer 节点生效 | 否        |
