row_purge_thread_count 
===========================================

`row_purge_thread_count` 用于设置执行内存中行内数据合并的最大工作线程数。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 4         |
| 取值范围             | \[1, 64\] |
| 是否重启 OBServer 节点生效 | 否         |
