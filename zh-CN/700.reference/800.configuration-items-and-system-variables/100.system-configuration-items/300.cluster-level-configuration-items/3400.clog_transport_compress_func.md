clog_transport_compress_func 
=================================================

`clog_transport_compress_func` 用于设置事务日志内部传输时使用的压缩算法。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>
