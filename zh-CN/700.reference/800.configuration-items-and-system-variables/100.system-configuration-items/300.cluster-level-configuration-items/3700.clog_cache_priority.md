clog_cache_priority 
========================================

`clog_cache_priority` 用于设置事务日志占用缓存的优先级。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 整型       |
| 默认值              | 1        |
| 取值范围             | \[1, +∞) |
| 是否重启 OBServer 节点生效 | 否        |
