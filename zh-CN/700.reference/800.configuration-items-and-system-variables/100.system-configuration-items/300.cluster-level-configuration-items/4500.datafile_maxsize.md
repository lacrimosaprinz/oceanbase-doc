# datafile_maxsize

`datafile_maxsize` 用于设置磁盘文件自动扩容的空间最大值。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从 V3.1.5 版本开始引入。</p>
</main>

| 属性 | 描述 |
| --- | --- |
| 参数类型 | 容量类型 |
| 默认值 | 0 |
| 取值范围 | [0M, +∞) |
| 是否重启 OBServer 节点生效 | 否 |
**示例**

```sql
alter system set datafile_maxsize = '10G';
```
