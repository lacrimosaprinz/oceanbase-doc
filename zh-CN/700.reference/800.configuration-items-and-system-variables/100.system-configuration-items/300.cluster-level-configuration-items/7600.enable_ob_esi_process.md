# enable_ob_esi_process

`enable_ob_esi_process` 用于控制是否启用 obesi（外部存储接口）进程。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

| **属性** | **描述** |
| --- | --- |
| 参数类型 | 布尔类型 |
| 默认值 | False |
| 取值范围 | <li>True：启用<li>False：禁用 |
| 是否重启 OBServer 节点生效 | 否 |
