# enable_pg

`enable_pg` 用于设置是否开启分区组功能。


<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|        属性        |                                                   描述                                                   |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | False                                                                                                  |
| 取值范围             | </li><li> True   </li><li> False    |
| 是否重启 OBServer 节点生效 | 否                                                                                                      |
