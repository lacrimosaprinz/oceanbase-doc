# enable_record_trace_log 

`enable_record_trace_log` 用于设置是否记录追踪日志。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                |
| 默认值              | True               |
| 取值范围             | <li> True   <li> False    |
| 是否重启 OBServer 节点生效 | 否                   |
