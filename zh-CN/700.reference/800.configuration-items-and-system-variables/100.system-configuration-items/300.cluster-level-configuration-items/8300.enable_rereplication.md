enable_rereplication 
=========================================

enable_rereplication 用于设置是否开启自动补副本的功能。


|        属性        |                                                   描述                                                   |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                |
| 默认值              | True                |
| 取值范围             | <li> True   <li> False    |
| 是否重启 OBServer 节点生效 | 否                   |
