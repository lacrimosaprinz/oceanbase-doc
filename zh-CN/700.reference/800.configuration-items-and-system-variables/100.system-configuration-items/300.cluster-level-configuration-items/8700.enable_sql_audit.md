enable_sql_audit 
=====================================

enable_sql_audit 用于设置是否开启 SQL 审计功能。


|      **属性**      |                                                    **描述**                                                     |
|------------------|---------------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                       |
| 默认值              | True                       |
| 取值范围             | <li> True：开启   <li> False：不开启    |
| 是否重启 OBServer 节点生效 | 否                          |
