backup_concurrency 
=======================================

backup_concurrency 用于设置备份时写文件系统的并发数量。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 0          |
| 取值范围             | \[0, 100\] |
| 是否重启 OBServer 节点生效 | 否          |
