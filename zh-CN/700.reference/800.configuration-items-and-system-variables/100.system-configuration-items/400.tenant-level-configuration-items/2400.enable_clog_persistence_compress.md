enable_clog_persistence_compress 
=====================================================

`enable_clog_persistence_compress` 用于设置是否开启事务日志落盘压缩。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>


|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 字符串类型  |
| 默认值              | false  |
| 取值范围             | NULL   |
| 是否重启 OBServer 节点生效 | 否      |
