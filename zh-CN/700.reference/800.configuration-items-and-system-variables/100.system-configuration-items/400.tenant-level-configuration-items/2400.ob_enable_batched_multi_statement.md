# ob_enable_batched_multi_statement

ob_enable_batched_multi_statement 用于设置是否启用批处理功能的成组执行优化。

|      **属性**      |                                                    **描述**                                                     |
|------------------|---------------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                       |
| 默认值              | False                      |
| 取值范围             | <li> False：不启用   <li> True：启用    |
| 是否重启 OBServer 节点生效 | 否                          |
