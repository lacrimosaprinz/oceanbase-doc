# enable_early_lock_release

`enable_early_lock_release` 用于设置是否开启提前解行锁的功能。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始默认值由 False 调整为 True。</p>
</main>

| **属性** | **描述** |
| --- | --- |
| 参数类型 | 布尔类型 |
| 默认值 | True |
| 取值范围 |<ul><li>  True </li><li> False </li></ul>|
| 是否重启 OBSever 节点生效 | 否 |
