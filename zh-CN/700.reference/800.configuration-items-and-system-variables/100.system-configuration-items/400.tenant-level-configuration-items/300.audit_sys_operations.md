audit_sys_operations 
=========================================

audit_sys_operations 用于设置是否追踪系统用户的操作行为。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                |
| 默认值              | False               |
| 取值范围             | <li> False   <li> True    |
| 是否重启 OBServer 节点生效 | 否                   |
