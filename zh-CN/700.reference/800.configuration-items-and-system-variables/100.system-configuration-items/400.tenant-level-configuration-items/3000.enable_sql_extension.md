enable_sql_extension 
=========================================

`enable_sql_extension` 用于设置是否开启租户的 SQL 扩展功能。


|      **属性**      |                                                    **描述**                                                     |
|------------------|---------------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                       |
| 默认值              | False                      |
| 取值范围             | <li> True：开启   <li> False：不开启    |
| 是否重启 OBServer 节点生效 | 否                          |


  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>该配置项当前仅在 MySQL 模式下生效。</p>
  </main>
