audit_trail 
================================

audit_trail 用于设置是否启用数据库审核。


|        属性        |                                   描述                                    |
|------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串                                                                     |
| 默认值              | NONE                                                                    |
| 取值范围             | <li> NONE   <li> OS   <li> DB   <li> DB，EXTENDED   <li> DB_EXTENDED    |
| 是否重启 OBServer 节点生效 | 否                                                                       |
