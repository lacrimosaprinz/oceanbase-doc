# query_response_time_flush

`query_response_time_flush` 用于设置是否刷新 `information_schema.QUERY_RESPONSE_TIME` 视图并重新读取 `query_response_time_range_base`。

| **属性** | **描述** |
| --- | --- |
| 参数类型 | 布尔类型 |
| 默认值 | False |
| 取值范围 | <li>True：启用<li>False：禁用 |
| 是否重启 OBServer 节点生效 | 否 |
