# rpc_memory_limit_percentage

`tenant_rpc_memory_limit_percentage` 用于设置租户中 RPC 的最大内存，占租户总内存的百分比。
默认值为 0 表示对 RPC 内存没有限制。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.1.0 版本开始引入。</p>
</main>

| **属性** | **描述** |
| --- | --- |
| 参数类型 | 整型 |
| 默认值 | 0 |
| 取值范围 | \[0,100] |
| 是否重启 OBSever 节点生效 | 否 |
