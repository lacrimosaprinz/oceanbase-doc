# 系统变量总览

## Global 系统变量

### 组复制插件相关

| 变量名                                                   | 功能描述                                                      |
|---------------------------------------------------------|---------------------------------------------------------------|
| group_replication_allow_local_disjoint_gtids_join       | 是否允许具有组中不存在的本地事务的服务器加入组。默认值为 `OFF`。  <main id="notice" type='explain'><h4>说明</h4><p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效，会产生 Warning。</p></main>    |
| group_replication_allow_local_lower_version_join        | 是否允许主版本低于组的服务器加入组。默认值为 `OFF`。<main id="notice" type='explain'><h4>说明</h4><p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效，会产生 Warning。</p></main>           |
| group_replication_auto_increment_increment              | 为在当前服务器实例上执行的事务指定连续列值之间的间隔时间。默认值为 `7`。 <main id="notice" type='explain'><h4>说明</h4><p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效，会产生 Warning。</p></main>       |
| group_replication_bootstrap_group                       | 是否配置当前服务器来 Bootstrap 组。默认值为 `OFF`。<main id="notice" type='explain'><h4>说明</h4><p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效，会产生 Warning。</p></main>    |
| group_replication_components_stop_timeout                | 组复制关闭时，等待每个组件的超时时间，单位为秒。默认值为 `2`。  <main id="notice" type='explain'><h4>说明</h4><p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效，会产生 Warning。</p></main>    |
| group_replication_compression_threshold                  | 组成员之间发送消息的阈值。当超过该阈值后，将会对组成员之间发送的消息进行压缩。默认值为 `0`，表示禁用压缩。 <main id="notice" type='explain'><h4>说明</h4><p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效，会产生 Warning。</p></main>  |
| group_replication_enforce_update_everywhere_checks       | 在任何时候启用或禁用多主更新的严格一致性检查。默认为 `OFF`，表示禁用检查查。 <main id="notice" type='explain'><h4>说明</h4><p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效，会产生 Warning。</p></main>    |
| group_replication_exit_state_action                      | 设置当服务器实例因异常而退出组时，组复制的行为方式。默认值为 `READ_ONLY`。 <main id="notice" type='explain'><h4>说明</h4><p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效，会产生 Warning。</p></main>  |
| group_replication_flow_control_applier_threshold         | 指定触发流量控制的应用程序队列中等待事务的个数。默认值为 `25000`。  <main id="notice" type='explain'><h4>说明</h4><p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效，会产生 Warning。</p></main>   |
| group_replication_flow_control_certifier_threshold       | 指定触发流控制的验证者队列中的等待事务的个数。 默认值为 `25000`。  <main id="notice" type='explain'><h4>说明</h4><p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效，会产生 Warning。</p></main>   |
