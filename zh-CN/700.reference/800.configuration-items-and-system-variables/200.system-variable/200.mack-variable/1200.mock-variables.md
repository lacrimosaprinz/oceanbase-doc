# 系统变量总览

## Global 系统变量

### 字符集和字符串相关

| 变量名   | 功能描述  |
|----------|----------|
| default_collation_for_utf8mb4      | 用于设置 utf8mb4 字符集的默认字符序，其默认值为 `utf8mb4_general_ci`。 <main id="notice" type='explain'><h4>说明</h4><p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效。</p></main> |
