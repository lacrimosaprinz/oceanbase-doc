# 系统变量总览

## Global 系统变量

### InnoDB 存储引擎相关

| 变量名   | 功能描述  |
|----------|----------|
| information_schema_stats_expiry       | 用于设置服务器从存储引擎获取数据并替换缓存中数据的秒数，其默认值为 `86400`。 <main id="notice" type='explain'><h4>说明</h4><p>该变量仅用于兼容 MySQL 5.7，相关功能并不支持，可以查询及设置，设置行为不报错但不生效。</p></main> |
