# query_rewrite_enabled

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该变量从 V4.3.1 版本开始引入。</p>
</main>

## 功能描述

`query_rewrite_enabled` 用于是否开启物化视图改写功能。

## 属性说明

| **属性** |  **描述**    |
|----------|--------------|
| 参数类型   |  enum                    |
| 默认值     |  true                  |
| 取值范围   |  [true，force，false] <ul><li>true：启用物化视图改写。</li><li>force：启用改写并跳过代价检查，强制使用匹配上的物化视图。</li><li>false：禁用物化视图改写。</li></ul>       |
| 生效范围   | <ul><li>GLOBAL</li><li>SESSION</li></ul>    |
| 是否可修改  | 是，支持通过 `SET` 语句修改。|

## 使用说明



## 配置示例

设置开启物化视图改写功能。

* Session 生效

  ```shell
  obclient> SET  query_rewrite_enabled= true;
  ```

* Global 生效

  ```shell
  obclient> SET GLOBAL  query_rewrite_enabled= true;
  ```
