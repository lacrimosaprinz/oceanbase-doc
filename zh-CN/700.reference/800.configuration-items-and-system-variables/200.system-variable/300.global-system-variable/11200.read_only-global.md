# read_only

read_only 用于设置租户是否为只读模式。

| **属性** |                                                     **描述**                                                      |
|--------|-----------------------------------------------------------------------------------------------------------------|
| 参数类型   | bool                         |
| 默认值    | 0                            |
| 取值范围   | <li> 1：租户为只读模式   <li> 0：租户为读写模式    |
| 生效范围   | GLOBAL                       |

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>设置后，会影响该租户下的所有普通用户，不影响 root 权限用户。</p>
  </main>
