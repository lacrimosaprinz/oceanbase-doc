# character_set_system

`character_set_system` 用于查看服务器使用的字符集。

| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | string                  |
| 默认值    | utf8mb4                 |
| 取值范围   | N/A                     |
| 生效范围   | <li> Global   <li> Session    |
| 是否可以修改 | 否                       |
