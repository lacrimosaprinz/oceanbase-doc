# version

`version` 用于查看服务器的版本号。

| **属性** | **描述**  |
|--------|---------|
| 参数类型   | varchar |
| 默认值    | 5.7.25-OceanBase-v4.0.0.0  |
| 取值范围   | <li>取值为空字符串时，返回值为默认值<li>取值为非空字符串时，返回值为设定值    |
| 生效范围   | Global  |
| 是否可修改  | 是       |

将版本号设置成默认版本号，示例如下：

```sql
obclient> SET GLOBAL version='';
```
