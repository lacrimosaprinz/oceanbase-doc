# character_set_database

`character_set_database` 用于设置默认数据库的字符集。

| **属性** |                                **描述**                                 |
|--------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型   | string                                                                |
| 默认值    | utf8mb4                                                               |
| 取值范围   | <li> utf8mb4   <li> binary   <li> gbk   <li> gb18030   <li> utf16    |
| 生效范围   | <li> Global   <li> Session                                                  |
| 是否可以为空 | 是                                                                     |
