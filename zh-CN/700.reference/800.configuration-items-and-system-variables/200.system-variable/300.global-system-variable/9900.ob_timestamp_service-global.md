ob_timestamp_service 
=========================================

`ob_timestamp_service` 用于指定使用何种时间戳服务。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

| **属性**  |                                                                     **描述**                                                                      |
|---------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型    | enum                                                                                                                                            |
| 默认值     | GTS                                                                                                                                             |
| 取值范围    | <li> LTS：事务执行过程中使用服务器本地的时间戳做为事务的快照，不支持跨分区的访问   <li> GTS：事务执行过程中从 GTS 服务获取全局时间戳做为事务的快照以及事务的提交版本号，使用上没有限制   <li>（暂不支持）HA_GTS    |
| 生效范围    | GLOBAL                                                                                                                                          |
| 是否参与序列化 | 是                                                                                                                                               |
