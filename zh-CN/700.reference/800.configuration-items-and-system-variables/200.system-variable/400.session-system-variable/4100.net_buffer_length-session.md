# net_buffer_length

`net_buffer_length` 用于设置 OBServer 节点能接受的 SQL Query 的大小。

| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | int                     |
| 默认值    | 16384                   |
| 取值范围   | \[1024, 1048576\]       |
| 生效范围   | <li> Global   <li> Session    |
| 是否可修改  | 该变量可通过 `SET GLOBAL` 语句修改 Global 生效方式下的取值，不可通过 `ALTER SESSION SET` 语句修改 Session 生效方式下的取值。                  |
