# ob_proxy_partition_hit

`ob_proxy_partition_hit` 用于设置 obproxy 是否将 SQL 语句请求的数据发送到这些数据主 Partition 所在的 OBServer 节点上。

| **属性** |                                               **描述**                                                |
|--------|-----------------------------------------------------------------------------------------------------|
| 参数类型   | bool             |
| 默认值    | 1                |
| 取值范围   | <li> 1：是   <li> 0：否    |
| 生效范围   | Session          |
| 是否可修改  | 该变量支持在创建租户时指定，租户创建后不能通过 SQL 语句修改                                                                    |
| 是否可见   | 否                |
